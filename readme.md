## Installing

This is a vundle/pathogen plugin, so first install that.

```
mkdir -p ~/.vim/bundle
cd ~/.vim/bundle
git clone https://github.com/VundleVim/Vundle.vim.git
```

Then you'll need to add this line to your .vimrc:

```
Plugin 'souffle-lang/souffle.vim'
```
